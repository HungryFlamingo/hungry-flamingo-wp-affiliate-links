Get advanced functionality for affiliate links and finally master affiliate-based monetization of your blog or website. Add beautiful and recognizable affiliate links automatically based on link target. 

== Description ==

This plugin helps you with monetizing your website. Automatically convert hyperlinks across your website into beautiful and effective affiliate ads.

== Features ==

= Automatic affiliate ad generation =

Add affiliate URLs to automatically be filtered. Add an icon and let the plugin automatically convert your hyperlinks into beautiful and effective affiliate ads.

= Minimum Requirements =
